# autoparts-marketplace-pakistan
Full-stack car parts marketplace for Pakistan - React + Node.js + MongoDB

## Problem
In Pakistan, finding specific auto parts (LED bulbs, engines, injectors, gaskets, etc.) is difficult because everything is offline.

## Solution
An online marketplace where shop owners can list their parts and customers/workshops can buy easily.

## Features
- Buyer and Seller roles
- Product listing with images
- Shopping cart and orders
- Separate dashboards
- Authentication system

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: MongoDB

## How to Run

Backend:
cd server
npm install
npm run dev

Frontend:
cd client
npm install
npm run dev

## Project Structure
client/ → React frontend
server/ → Node.js backend
